# Demo App: Hello-World UI

A simple web app to demonstrate
[`@spautz/basic-library-template`](https://github.com/spautz/package-template/blob/main/packages/basic-library-template/) and
[`@spautz/react-library-template`](https://github.com/spautz/package-template/blob/main/packages/react-library-template/)
(the packages in this repo).

In-workspace demo apps are used:
* To demo and document the package from a consumer's perspective
* For CI checks, exercising the package from outside to ensure it works properly
* As a harness for local dev work
* As a potential deployment for a live public demo
